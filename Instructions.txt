*** Fix Stupid PowerSchool attendance page UI problems (SDUSD) ***

a. Widen the name column so names don't wrap onto a second line that 
   makes the attendance page really long
b. One click function to mark a student absent (instead of three)

*** Installation instructions ***

1. Open Google Chrome
2. Enter chrome://extensions for the URL
3. Enable Developer mode (top right corner)
4. Click Load unpacked extension...
5. Choose the ExtensionFiles folders located in the same folder as 
   these instructions.
6. The extension will load into Chrome
7. Uncheck Developer mode box
8. In the PowerSchool Fix Attendance Columns box click Details
9. Check Allow in Incognito

You can delete the files when you're finished installing the extension

*** Operation ***

1.  Now when you go to any attendance page, the column width for the student name 
    will be wide enough to display almost all student names on one line!
2.  Click once on the drop down to mark a student absent.  Click again to change the
    setting to Present or Tardy
3.  You can disable or delete this extension any time by going to chrome://extensions
